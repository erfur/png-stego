PNG (Pronounced as "ping")
----------------------------

	[File Structure]

		The bytes are in network order.

		First eight bytes:
			137 80 78 71 13 10 26 10

		Need an IHDR chunk in the beginning, and an IEND chunk in the end.

		[Chunk structure]
			uint_32_t datafield_size
			uint_32_t chunk_type
			?? chunk_data
			uint_32_t crc

		[Critical chunk types]

			IHDR: Image Header chunk
				width				4 bytes
				height				4 bytes
				bit depth			1 byte
				color type			1 byte
				compression method	1 byte
				filter method		1 byte
				interlace method	1 byte
									25 bytes total

			IDAT: Image Data chunk
				zlib data blocks compressed with deflate

			IEND: End of file chunk 
				Stored empty (12 bytes?)

			zlib format
				Compression method/flags code	1 byte
				Additional flags/check bits		1 byte
				Compressed data blocks			n bytes
				Check value						4 bytes
